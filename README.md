# Dotfiles - Death Note Rice

## Setup
- WM: Hyprland
- Bar: Waybar
- Terminal: Kitty
- Shell: ZSH + Oh My Zsh + Starship
- Launcher: Wofi
- Editor: Neovim
- File Manager: Yazi + Thunar
- Music: MPD + ncmpcpp
- Fetch: Fastfetch
- Theme: Dark Red / Death Note

## Instalar
\`\`\`bash
git clone https://github.com/sarvire/dotfiles
cd dotfiles
cp -r * ~/.config/
